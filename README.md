## Manual Setup
1. Download mit Git:
```bash
git clone https://github.com/MariaHildebrandt/MEAN-ToDo-App projectname
```
2. navigate to projectname and install with node package manager
```bash
npm install
```
3. start your project and open http://localhost:3000 in your browser
```bash
npm start
```
